# Node.js CRUD Uygulaması

Bu proje, AWS Lambda, DynamoDB, AWS Amplify ve AWS Cognito'yu kullanarak güvenli ve ölçeklenebilir bir CRUD (Create, Read, Update, Delete) uygulaması oluşturulmuştur.

### Gereksinimler
- Node.js ve npm
- AWS CLI
- Amplify CLI
